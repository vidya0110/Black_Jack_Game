# 🎴 Blackjack Game

A modern browser-based Blackjack game built using **HTML, CSS, and JavaScript**.  
This project simulates a real casino-style experience with smooth UI, animations, and game logic.

---

## 🚀 Features
✨ Interactive gameplay (Hit / Stand)  
🎯 Dealer AI (draws cards until reaching 17)  
🧠 Smart Ace handling (1 or 11 automatically)  
🎨 Clean UI with gradient background and animations  
🎴 Dynamic card rendering  
🔄 Restart game functionality  

---

## 🛠 Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 🎮 How to Play
1. Click **Hit** to draw a card  
2. Click **Stand** to end your turn  
3. Try to get as close to **21** as possible without going over  
4. Beat the dealer to win 🏆
